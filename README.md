# VentasHDL
## Repo donde iré almacenando el sw "Ventas HDL"

La idea es desarrollar una app sencilla de ventas.

Deberá:
- Almacenar una venta
- La venta tendrá diferentes productos
- La venta pertenecerá a un cliente
- Enviar un resumen por correo
